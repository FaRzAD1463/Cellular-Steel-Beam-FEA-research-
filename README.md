Cellular Steel Beam FEA
Overview

This project investigates the structural behavior of cellular steel beams using Finite Element Analysis (FEA). Cellular beams are steel beams with circular openings in the web, which reduce weight while maintaining structural efficiency.

The goal of this research is to analyze stress distribution, deformation, and load capacity under different loading and geometric conditions.

The study also integrates Python automation to assist in simulation setup, model modification, and data extraction for efficient analysis.

Objectives

Study the mechanical behavior of cellular steel beams

Analyze stress concentration around web openings

Evaluate deformation and load-bearing capacity

Compare different geometric parameters

Support structural design optimization using simulation data

Methodology

The research uses Finite Element Analysis (FEA) to simulate structural performance under various conditions.

The workflow includes:

Creating the beam geometry with cellular openings

Assigning material properties

Applying boundary conditions and loads

Running simulations

Extracting and analyzing structural responses

Python scripts are used to automate parts of the simulation process, including model configuration and data extraction.

Technologies Used

Python

ABAQUS

Finite Element Analysis (FEA) methods

Numerical simulation and data processing

Key Analysis Parameters

The simulations consider several structural parameters, including:

Beam geometry

Diameter and spacing of cellular openings

Material properties

Load conditions

Boundary conditions

These parameters help evaluate how design variations affect structural performance.

Expected Results

The simulations aim to provide insights into:

Stress concentration around openings

Deflection patterns

Structural efficiency of cellular beams

Optimal design configurations

The results can support better structural design and optimization of cellular steel beams
